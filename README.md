# BioProtienDB
A SQL database for genes, proteins and diseases.
BioProteinDB

Overview

BioProteinDB is a relational SQL database developed as a biotechnology portfolio project. It stores information about organisms, genes, proteins, biological functions, and diseases while demonstrating database design principles.

Technologies

- MySQL
- MySQL Workbench
- SQL

Database Tables

- Organism
- Gene
- Protein
- Function_Table
- Disease
- Protein_Function
- Protein_Disease

Features

- Relational database design
- Primary and foreign keys
- Many-to-many relationships
- Sample biological data
- SQL queries for biological analysis

Skills Demonstrated

- SQL
- Database normalization
- Relational database design
- Data modeling
- Biotechnology data organization

Future Improvements

- Add more genes and proteins
- Import larger public datasets
- Add pathways and protein interactions
- Build a dashboard using Power BI or Tableau
